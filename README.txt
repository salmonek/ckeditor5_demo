INTRODUCTION
------------

This is a simple module to show basic CKEditor 5 capabilities on a Drupal site.


REQUIREMENTS
------------

No special requirements.


CONFIGURATION
-------------

Configure the module at (/admin/config/ckeditor5-demo).
You'll need only to provide an url to the ckeditor.js file.
Currently module only supports classic version of CKEditor 5.
